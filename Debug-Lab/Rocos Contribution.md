## Contribute

### 前置知识

1. [Git 基础使用](https://www.runoob.com/git/git-tutorial.html): Commit 流程，Git 与 GitHub 的结合使用， 以及分支管理

2. [Git Angular Commit 规范](https://juejin.cn/post/7126022242508472351)

### 贡献你的代码

1. Fork 分支到你的仓库

![1](https://gitee.com/LanternCX/picx-images-hosting/raw/master/2025-03-11/1.6t79qckfrc.webp)

![2](https://gitee.com/LanternCX/picx-images-hosting/raw/master/2025-03-11/2.pfhnwhrd5.webp)

2. Clone 仓库到本地

![3](https://gitee.com/LanternCX/picx-images-hosting/raw/master/2025-03-11/3.9kgbyf6jt5.webpg)

![4](https://gitee.com/LanternCX/picx-images-hosting/raw/master/2025-03-11/4.8vn2eej0sq.webp)

3. 在 VSCode 中打开项目并切换到 `dev` 分支

![5](https://gitee.com/LanternCX/picx-images-hosting/raw/master/2025-03-11/5.sz3lmau2y.webp)

4. 提交 commit

注意 commit 消息要遵守 Angular Commit 规范，说明进行了何种改动以及改动了什么部分

![6](https://gitee.com/LanternCX/picx-images-hosting/raw/master/2025-03-11/6.8l08l93snf.webp)

![7](https://gitee.com/LanternCX/picx-images-hosting/raw/master/2025-03-11/7.26lmpnlw3v.webp)

4. Push commit 到你的 GitHub 仓库并创建 PR(Pull Request)

![8](https://gitee.com/LanternCX/picx-images-hosting/raw/master/2025-03-11/8.2a58ndeyto.webp)
