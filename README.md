# Caoxin Blog

基于 Hexo 的个人博客站点，用于整理技术文章、项目记录与个人内容。

## 快速开始

1. 运行前提：本地已安装 Node.js 与 npm
2. 安装依赖：`npm install`
3. 运行测试：`npm test`
4. 迁移样本内容并刷新映射说明：`npm run migrate:samples`
5. 清理旧的 Hexo 产物：`npm run hexo:clean`
6. 生成静态文件：`npm run hexo:generate`
7. 本地预览站点：`npm run hexo:serve`

推荐的首次验收顺序如下：

```bash
npm install
npm test
npm run migrate:samples
npm run hexo:generate
npm run hexo:serve
```
